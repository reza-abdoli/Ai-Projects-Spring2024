Main form of simple linear regression function: $$f(x) = \alpha x + \beta$$

here we want to find the bias ($\alpha$) and slope($\beta$) by minimizing the derivation of the Residual Sum of Squares (RSS) function:

  • step 1: Compute RSS of the training data
$$ RSS = \Sigma (y_i - (\hat{\beta} + \hat{\alpha} * x_i) )^2 $$
  • step 2: Compute the derivatives of the RSS function in terms of $\alpha$ and $\beta$, and set them equal to 0 to find the desired parameters
$$ \frac{\partial RSS}{\partial \beta} = \Sigma (-f(x_i) + \hat{\beta} + \hat{\alpha} * x_i) = 0$$$$ \to \beta = \hat{y} - \hat{\alpha} \hat{x} \to (1)$$$$ \frac{\partial RSS}{\partial \alpha} = \Sigma (-2 x_i y_i + 2 \hat{\beta} x_i + 2\hat{\alpha} x_i ^ 2) = 0 \to (2)$$$$ (1) , (2) \to \hat{\alpha} = \frac{\Sigma{(x_i - \hat{x})(y_i - \hat{y})}}{\Sigma{(x_i - \hat{x})^2}} $$$$ \hat{\beta} = y - \hat{a} x$$

def linear_regression(input, output):
    x_mean = np.mean(input)
    y_mean = np.mean(output)
    xy_covariance = np.mean(input * output) - x_mean * y_mean
    x_variance = np.mean(input ** 2) - x_mean ** 2

    alpha = xy_covariance / x_variance
    beta = y_mean - alpha * x_mean

    return alpha, beta

    squared_errors = np.power(actual_values - predicted_values, 2)  
    sum_of_squared_errors = np.sum(squared_errors)

The RMSE has no bound, thus it becomes challenging to determine whether a particular RMSE value is considered good or bad without any reference point. Instead, we use R2 score. The R2 score is calculated by comparing the sum of the squared differences between the actual and predicted values of the dependent variable to the total sum of squared differences between the actual and mean values of the dependent variable. The R2 score is formulated as below:

$$R^2 = 1 - \frac{SSres}{SStot} = 1 - \frac{\sum_{i=1}^{n} (y_{i,true} - y_{i,pred})^2}{\sum_{i=1}^{n} (y_{i,true} - \bar{y}_{true})^2} $$

def get_r2_score(predicted_values, actual_values):
    residual_sum_of_squares = np.sum(np.power(actual_values - predicted_values, 2))
    
    total_sum_of_squares = np.sum(np.power(actual_values - np.mean(predicted_values), 2))
    
    r2_score_val = 1 - (residual_sum_of_squares / total_sum_of_squares)
    
    return r2_score_val

Ploynomial Regression

To extend the simple linear regression to polynomial regression, we can model the relationship between the independent variable $x$ and the dependent variable $y$ as a polynomial function of degree $n$:

$$f(x) = \beta_0 + \beta_1x + \beta_2x^2 + \ldots + \beta_nx^n$$

The steps to find the parameters $\beta_i$ are similar to those in simple linear regression. We again minimize the RSS function by taking the derivatives with respect to each parameter and setting them to 0.

  • Step 1: Compute the RSS function for polynomial regression:
$$ RSS = \Sigma (y_i - (\hat{\beta_0} + \hat{\beta_1}x_i + \hat{\beta_2}x_i^2 + \ldots + \hat{\beta_n}x_i^n))^2 $$
  • Step 2: Compute the derivatives of the RSS function with respect to each parameter $\beta_i$ and set them to 0 to find the desired parameters.
$$ \frac{\partial RSS}{\partial \beta_i} = 0, \text{ for } i = 0, 1, 2, \ldots, n$$

Solving these equations will give us the optimal values of $\beta_i$ for the polynomial regression model. The specific form of the equations will depend on the degree of the polynomial and the number of parameters.

The general form for finding the coefficients for polynomial regression can be represented as:

$$ \beta = (X^T X)^{-1} X^T y $$

where:

  • $X$ is the design matrix with columns $x^0, x^1, x^2, ..., x^n$
  • $x^i$ represents the feature vector of $x$ raised to the power of $i$
  • $y$ is the target variable vector
  • $\beta$ is the coefficient vector for the polynomial regression

By solving for $\beta$ using the above formula, we can obtain the coefficients for the polynomial regression model.

Q32

In random forests, several independent decision trees are trained simultaneously. Each tree is randomly constructed from a subset of data and features. Then, at each step, as new data comes in, each tree makes its own prediction, and their results are combined to make a final prediction. This combination can be in the form of majority voting (which predicts the highest number of votes) or the average of the values predicted by each tree. Because each tree is trained independently and is made from a random sample of data, this method benefits from increasing flexibility and reducing overfitting.

Q32

In random forests, bootstrapping is an important process used to build each decision tree. This method employs the resampling with replacement technique.

Bootstrapping steps are as follow:

  1. Backward sampling: New samples are generated from the existing data set. In this process, each data point may be repeated several times or may not be present in the formation of a new samle.

  2. Decision tree training: Using the new samples from the previous step, a decision tree is trained. This tree is randomly constructed from the dataset and may use different variables of the samples.

  3. Combination of results: Finally, by applying the voting or averaging technique on the predictions of each tree, the final result is otained.

Bootstrapping creates more flexibility in the model due to the generation of random and multiple samples. This method prevents overfitting and increases the accuracy and performance of the model in the test data. Additionally, this method can help the model better withstand outliers and noisy data.

Q33

Bootstrapping is used in random forests to train multiple decision trees by generating random samples from the available data and combine their results. This prevents overfitting and improves model performance. Bootstrapping increases the model's flexibility and resistance to outliers and noise.
